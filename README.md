# Run the server

```npm run server```


# Start the app
```npm start```


# Features

 -  List Docker Containers: Displays all available Docker containers, including their names, IDs, and status.
  - Container Start/Stop: Allows starting or stopping containers with a toggle button, depending on their current state.
  - Inspect Container: Provides detailed information about the selected container, such as configuration, state, and created time, in a modal view.
  - View Logs: Displays real-time logs of the selected container in a dedicated modal.
  - View Container Stats: Shows container statistics, including CPU usage, memory usage, and network I/O.
  - View Services within a Container:
     - Lists services running within the container (if supported).
     -  Allows starting and stopping services directly from the UI.
